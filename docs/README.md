## 🎯 기능 요구사항

- 덧셈 기능

  - [x] 2개의 숫자에 대해 덧셈이 가능하다.
  - [x] [예외] 1개의 숫자를 입력 시 경고창(alert 문)을 띄어준다.
  - [x] [예외] 입력된 숫자가 없을 시 경고창(alert 문)을 띄어준다.
  - [x] [예외] 숫자가 3자리를 초과할 경우 경고창(alert 문)을 띄어준다.
